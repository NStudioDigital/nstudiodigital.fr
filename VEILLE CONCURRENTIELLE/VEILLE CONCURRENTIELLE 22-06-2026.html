Analyse stratégique

⚠️ Concurrents à surveiller de près

  • → Agence Dilo : même niche (instituts beauté), même prix d'entrée (990 €) mais pour un one-page seulement. Site complet à 1 570 €. Nationale. Surveiller leur positionnement SEO sur les requêtes "site coiffeur Lyon".
  • → Digital Unicorn : concurrent local Lyon, généraliste mais 3 semaines de délai identiques. À 1 500 €. Leur SEO est fort sur "agence web Lyon".
  • → Fabrik Web / Wasabi Web : acteurs nationaux spécialisés coiffeurs, bien référencés sur les requêtes longue traîne. Leurs blogs captent le trafic éducatif.

✅ Vos avantages concurrentiels réels

  • → 990 € pour 5 pages custom : unique sur le marché Lyon. La concurrence locale est à 1 500 €+, la niche beauté nationale à 1 570 €+ pour le même périmètre.
  • → Code custom (non WordPress) : 0 dépendance plugin, 0 risque de hack WordPress. Argument de fond différenciateur.
  • → Propriété totale client : GitHub + Vercel au nom du client. Rarissime à ce prix.
  • → Ancrage Grand Lyon : aucun concurrent local ne fait la niche beauté/bien-être. Vous seule combinez localisation + spécialisation.
  • → Maintenance à 19 €/mois sans engagement : le moins cher observé (concurrents à 30–80 €/mois).

🔎 Points de vigilance

  • → ReflexPrinting à 390 € : peut séduire les prospects très price-sensitive. Argument de réponse : pas de propriété, hébergement mutualisé, pas de code propre.
  • → Linkeo / Rdv360 : abonnement mensuel qui semble moins cher à court terme. Mais lockin total, 0 propriété, 100–1 200 €/an en permanence.
  • → WordPress est la norme : la plupart des concurrents livrent sur WordPress. Certains prospects pourraient demander "pourquoi pas WordPress ?". Préparer la réponse (vitesse, sécurité, propriété).
  • → Agence Dilo monte en SEO : surveiller si elle rankera sur "site web coiffeur Lyon" dans les mois à venir.

💡 Opportunités identifiées

  • → Aucun acteur local niche : le marché Lyon est non adressé par des spécialistes beauté/bien-être. C'est votre moat principal.
  • → SEO long traîne : les concurrents nationaux (Fabrik Web, Wasabi) captent "site coiffeur France" mais pas "site web coiffeur Villeurbanne" ou "création site head spa Lyon". Votre blog peut dominer ces requêtes locales.
  • → Praticiens bien-être (sophro, naturo, yoga) : segment très peu servi par les concurrents identifiés. Vos pages dédiées sont uniques.
  • → Différenciation "code pur" : valoriser explicitement le no-WordPress dans vos arguments de vente et sur votre site.
Recommandations actionnables
1

Surveiller Agence Dilo chaque trimestre

Vérifier leur positionnement Google sur "site web coiffeur Lyon" et "site institut beauté Lyon". Si elles commencent à ranker localement, accélérer votre production de contenu local.

Impact : protection du territoire
2

Créer une page "Pourquoi pas WordPress ?" sur votre site

La majorité des concurrents sont sur WordPress. Une page dédiée (ou section FAQ) expliquant les avantages du code custom vous différencie immédiatement et répond à l'objection avant qu'elle ne soit posée.

Impact : conversion prospect + SEO
3

Construire un argumentaire "prix vs concurrence" clair

990 € pour 5 pages custom = moins cher qu'Agence Dilo (1 570 €), Digital Unicorn (1 500 €) et tous les freelances spécialisés beauté. À mentionner dans la page tarifs et les cold emails. "Moins cher que les spécialistes nationaux, ancré dans votre ville."

Impact : closing commercial
4

Cibler les praticiens bien-être comme angle différenciant principal

Sophrologues, naturopathes, yoga, pilates : aucun concurrent identifié ne les adresse spécifiquement avec un site web dédié. C'est votre niche dans la niche.

Impact : positionnement sans concurrence
5

Lancer une veille mensuelle automatisée (Google Alerts)

Configurer des alertes Google sur : "agence web coiffeur Lyon", "création site beauté Lyon", "site vitrine salon Lyon". Vous serez notifiée dès qu'un nouveau concurrent émerge ou qu'une agence existante se spécialise sur votre territoire.

Impact : surveillance continue (5 min/mois)
